# 프롬프트 마법사 🪄

AI(ChatGPT, Gemini, Claude 등)에 최적화된 프롬프트를 자동 생성해주는 웹앱입니다.

## 🚀 배포 방법 (Vercel)

### 1단계: GitHub에 올리기
```bash
git init
git add .
git commit -m "첫 배포"
git remote add origin https://github.com/본인아이디/prompt-wizard.git
git push -u origin main
```

### 2단계: Vercel 배포
1. https://vercel.com 접속 → GitHub 로그인
2. "Add New Project" → 위에서 만든 저장소 선택
3. "Deploy" 클릭 → 완료!
4. 생성된 URL로 핸드폰/PC 어디서나 접속 가능

## 📱 홈 화면 추가 (앱처럼 사용)
- **iPhone**: Safari에서 접속 → 공유 → 홈 화면에 추가
- **Android**: Chrome에서 접속 → 메뉴 → 앱 설치

## 💻 로컬 실행
```bash
npm install
npm run dev
```
